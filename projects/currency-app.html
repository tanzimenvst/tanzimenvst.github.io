What This App is ?

The Worldwide Exchange Rate Dashboard is a dynamic currency data visualization tool built using Streamlit. It provides users with up-to-date and historical exchange rate data for 180+ of global currencies, allowing for complex multi-currency comparisons and single-pair time-series analysis over the last 50 days.

This project showcases my capability to build a robust, integrated visualization platform for complex, multivariate time-series data. The application is designed to efficiently ingest and present multiple disparate data streams within a single, cohesive, and performant user interface.

Currency Comparison Tool

Users select a base and a target currency to view a dedicated line chart showing the exchange rate movement over the past 50 days. This offers a precise, focused trend analysis between two specific currencies.

Multi-Currency Performance

Users can select multiple currencies via continent-wise expanding sidebar menus. The application generates a single chart that normalizes and plots the performance of all selected currencies against the USD, allowing for rapid detection of appreciation/depreciation patterns.
