Generate DEM From XYZ Values in CSV Files

The objective of this script is to convert a large number of CSV files consists of coordinated along with elevation to raster data (DEM).

It is a pretty simple task. However, if your dataset consists of thousand of CSV files, you can't do that manually. It will take weeks. And if you plan to do that often, you need an automated way.

  • I used Pandas to read the CSV files and added headers if missing.
  • Instead of converting the coordinates into points, I used ArcPy to virtually load the coordinates into the map canvas.
  • I then used ArcPy again to convert the virtually loaded points into raster data.
  • Finally, I used GDAL to merge all the rasters into a single raster file.

Result: The script was able to make a single raster by combining 784 csv files in about 3 hours, where there was nearly 950 million points.

# Generating rasters from each CSV file and saving them to temp folder

for csv_file in csv_files:
    csv = os.path.splitext(os.path.basename(csv_file))[0]

    try:
        with open(csv_file, 'r+') as f:
            content = f.readlines()
            
            if not content:
                continue
            
            first_line = content[0].strip()
            first_value = first_line.split()[0] if first_line else ""

            if is_number(first_value):
                new_content = ["Field1 Field2 Field3\n"] + content
            else:
                new_content = ["Field1 Field2 Field3\n"] + content[1:]

            f.seek(0)
            f.writelines(new_content)
            f.truncate()
    except Exception as e:
        print(f"Failed to insert header to {csv}.csv: {str(e)}")
        sys.exit(1)

    try:
        arcpy.MakeXYEventLayer_management(table=csv_file, in_x_field="Field1", in_y_field="Field2", out_layer="csv_Layer", spatial_reference=spatial_reference, in_z_field="Field3")
    except Exception as e:
        print(f"Failed to make XY layer for {csv}.csv: {str(e)}")
        sys.exit(1)

    try:
        arcpy.FeatureToRaster_conversion(in_features="csv_Layer", field="Field3", out_raster=os.path.join(ras, f"{csv}.tif"), cell_size=cell_size)
    except Exception as e:
        print(f"Failed to generate Raster for {csv}.csv: {str(e)}")
        sys.exit(1)

# Merging all temp rasters to one DEM

dataset = [gdal.Open(tiff) for tiff in tiff_files]
warp_options = gdal.WarpOptions(format='GTiff', dstNodata=0)

try:
    gdal.Warp(os.path.join(output_path, "DEM.tif"), dataset, options=warp_options)
    print(f"{total_ras} rasters successfully merged into one and saved to: {output_path}\\DEM.tif")
except Exception as e:
    print(f"Failed to merge all temp rasters: {str(e)}")
    sys.exit(1)
