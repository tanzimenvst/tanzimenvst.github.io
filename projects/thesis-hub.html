Project Overview

A web-based GIS platform to spatially visualize, explore, and manage academic thesis research on an interactive map.

Project Objectives
  • Create a centralized platform to visualize academic research spatially.
  • Enable students and researchers to explore thesis study areas interactively.
  • Help universities map alumni research and identify unexplored locations.
  • Transform institutional research data into a discoverable GIS experience.
Key Features
  • Interactive Leaflet map displaying thesis locations using markers.
  • Filtering by university, department, degree, and roll number.
  • Detailed popup with thesis title, abstract, study area, and results.
  • User authentication with registration, login, and email verification.
  • Thesis upload system with spatial data visualization.
  • Admin CMS to verify submissions and manage datasets.
  • Automated email notifications for updates and actions.
Technologies Used
Flask (Python) HTML Bootstrap 5 JavaScript PostgreSQL PostGIS Leaflet GeoServer (WMS) Supabase Auth Render Hosting
System Readiness & End-to-End Implementation

This application has been developed as a fully functional, deployment-ready system, covering the complete workflow from data ingestion to visualization and user interaction. The platform is not a prototype anymore; now it represents a complete full-stack GIS solution with integrated frontend, backend, database, authentication, and deployment management.

  • Automated Data Upload & Processing: Users can submit thesis metadata and spatial study area information directly through the web interface. Uploaded data is validated, stored in a PostGIS-enabled database, and immediately available for map visualization without manual processing.
  • Complete User Account Management: Secure registration and login system with email verification, password reset functionality, and session management, ensuring a reliable and user-friendly authentication experience.
  • Admin-Controlled Content Management System (CMS): An administrative panel allows verification, approval, or rejection of submitted theses, ensuring data quality, authenticity, and controlled public access.
  • Automated Email Notification System: Users receive email notifications for account-related actions, thesis submission status, and administrative decisions, improving transparency and engagement.
  • Interactive GIS Visualization: Real-time spatial visualization using Leaflet and GeoServer (WMS), with advanced filtering by university, department, degree, and roll number, along with detailed popups containing thesis information.
  • Integrated & Scalable Architecture: The system integrates frontend UI, backend APIs, spatial databases, authentication services, and hosting infrastructure into a cohesive architecture designed for scalability and institutional deployment.
The platform is fully operational and ready for real-world academic or institutional use. The attached live exploration deployment utilizes free-tier hosting services, which may result in minor performance limitations under heavy load, and the mobile view is not yet fully supported.
